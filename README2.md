# Mini Log Analyzer

A Python script that analyzes a log file and counts log levels.

## Description
This project reads a log file and counts occurrences of:
- ERROR
- WARNING
- INFO

It also counts total lines in the file.

## Features
- File reading
- Log classification
- Simple text analysis
- Useful practical project

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Place your log file in the project folder
3. Run:

   python log_analyzer.py

4. Enter the log file name

## Example

Input:
sample.log

Output:
Total Lines: 5
ERROR: 2
WARNING: 1
INFO: 2
